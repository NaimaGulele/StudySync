# StudySync

A modern, multilingual study management application designed to help students organize tasks, track progress, and manage their learning resources efficiently.

## Features

✨ **Multi-Language Support**
- English
- Portuguese (Português)
- Changana (Xichangana)

📱 **Responsive Design**
- Mobile-first approach
- Optimized for phones, tablets, and desktops
- Touch-friendly interface
- Smooth animations and transitions

📋 **Task Management**
- Create, edit, and delete tasks
- Set priorities (High, Medium, Low)
- Track due dates and completion status
- Filter by subject and priority

📚 **Subject Organization**
- Organize tasks by subject/course
- Color-coded subjects
- Track progress per subject
- View completion statistics

🔗 **Resource Management**
- Save study resources (links, notes, PDFs, videos)
- Organize resources by subject
- Quick access to learning materials

👤 **User Profile**
- Customize your study profile
- Track personal statistics
- Manage study preferences

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Python 3 (for local development server)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/NaimaGulele/StudySync.git
cd StudySync
```

2. Start the development server:
```bash
npm start
# or
python3 -m http.server 3000
```

3. Open your browser and navigate to:
```
http://localhost:3000
```

## Usage

### Adding a Task
1. Click the "Add Task" button
2. Fill in the task name, select a subject
3. Set the due date and priority
4. Click "Save Task"

### Creating a Subject
1. Navigate to the Subjects page
2. Click "Add Subject"
3. Enter the subject name and select a color
4. Click "Add Subject"

### Adding Resources
1. Go to the Resources page
2. Click "Add Resource"
3. Enter the resource title, type, and URL
4. Click "Add Resource"

### Switching Languages
Click the language buttons in the sidebar (EN, PT, TS) to switch between English, Portuguese, and Changana.
