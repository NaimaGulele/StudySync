# StudySync Voice Input Guide

## How to Use Voice Input

Your StudySync app now has a powerful voice input feature that lets you create and manage tasks using your voice.

### Quick Voice Task Creation

The quickest way to add a task is using the voice input button:

1. **Click the Microphone Button (🎤)** - Located in the top right next to the "+ Add Task" button
2. **Wait for "Listening..." indicator** - The blue indicator at the bottom center shows the app is ready
3. **Speak your task clearly** - Say what you need to do, e.g.:
   - "Study Chapter 5 for HCI"
   - "Read Norman's design principles"
   - "Complete the algebra problem set"
4. **Task is automatically saved** - The app will:
   - Record your speech
   - Convert it to text
   - Create a new task with today's due date
   - Automatically assign it to your first subject
   - Show a confirmation message

### Voice Input in Task Form

You can also use voice input when adding a task manually:

1. **Click "+ Add Task"** - Opens the task creation form
2. **Click the Microphone (🎤)** - Next to the Task Name field
3. **Speak the task name** - The text will appear in the field
4. **Fill in other details manually** - Subject, due date, priority
5. **Click "Save Task"** - To save the complete task

## Voice Features

### Supported Browsers

Voice input works in:
- **Chrome/Edge** - Full support
- **Firefox** - Full support
- **Safari** - Full support (iOS 14.5+)
- **Mobile browsers** - Chrome, Firefox, Safari on iOS/Android

### What Gets Saved

When you use quick voice input, a new task is created with:
- **Task Name** - What you said (transcribed)
- **Due Date** - Today's date
- **Subject** - Your first/default subject
- **Priority** - Medium (can edit after)
- **Status** - Incomplete

### Visual Feedback

The app provides clear feedback:

1. **Voice Button States**
   - Normal: Blue circle with microphone
   - Recording: Red pulsing button with animation
   
2. **Voice Indicator**
   - Shows "Listening..." when recording
   - Animates with a pulsing dot
   - Disappears when not in use

3. **Confirmation**
   - Toast notification shows task was saved
   - Example: "✅ Task saved: Study Chapter 5 for HCI"

## Tips for Best Results

1. **Speak Clearly** - Use normal volume and pace
2. **Be Specific** - Include what subject/chapter if possible
3. **Short Phrases** - Keep task names brief and actionable
4. **Check Results** - Verify the transcription is correct

## Troubleshooting

### Voice Input Not Working

**Problem**: Voice button doesn't work or shows error
- **Solution 1**: Check browser support (Chrome/Firefox/Safari)
- **Solution 2**: Allow microphone permissions when prompted
- **Solution 3**: Check your microphone is working (test in browser settings)
- **Solution 4**: Refresh the page and try again

### No Microphone Permission

**Problem**: Browser asks for microphone permission
- **Solution**: Click "Allow" when the browser asks for microphone access
- Note: You only need to allow this once per device

### Unclear Transcription

**Problem**: The speech-to-text is incorrect
- **Solution 1**: Speak more slowly and clearly
- **Solution 2**: Use the task form with voice input, then edit the text
- **Solution 3**: Create the task manually without voice

## Accessibility

The voice feature is designed to be accessible:

- Can be used entirely with voice (hands-free)
- Visual indicators show when recording
- Keyboard controls available (voice button is focusable)
- Works on mobile and desktop
- Supports all three languages (EN, PT, CH)

## Privacy & Security

Your voice input is:
- Processed by your browser's Web Speech API
- Not sent to external servers (depends on browser)
- Immediately converted to text
- Used only to create tasks
- Never stored in recordings

## Keyboard Shortcut Alternative

No keyboard shortcut yet, but you can:
1. Use Tab to navigate to the voice button
2. Press Enter to activate it
3. Speak your task
4. Press Escape to stop recording
