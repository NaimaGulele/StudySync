# StudySync Quick Start Guide

## Installation

### Option 1: Run Locally with npm

```bash
# Install dependencies (already done)
npm install

# Start the dev server
npm run dev

# Open in browser
http://localhost:3000
```

### Option 2: Run with Python

```bash
# No npm needed
python3 -m http.server 3000

# Open in browser
http://localhost:3000
```

### Option 3: Deploy to Vercel

```bash
# Push to GitHub first
git add .
git commit -m "Deploy StudySync"
git push origin main

# Then deploy on vercel.com
vercel
```

## First Steps

### 1. View Dashboard

The dashboard shows you:
- Total tasks (5)
- Completed tasks (1)
- Due soon tasks (4)
- Total subjects (4)
- Today's tasks
- All pending tasks
- Upcoming deadlines
- Subject progress

### 2. Add Your First Task with Voice

1. Click the blue microphone button (🎤) in the top right
2. Wait for "Listening..." to appear
3. Say your task: "Study Chapter 3"
4. Watch it save automatically

### 3. Add a Task Manually

1. Click "+ Add Task"
2. Fill in the task name
3. Select the subject
4. Pick a due date
5. Choose priority (High, Medium, Low)
6. Click "Save Task"

### 4. Switch Language

Click EN, PT, or CH in the left sidebar to change language:
- **EN** - English
- **PT** - Portuguese (Português)
- **CH** - Changana (Mozambique)

### 5. Install as App

When you see "Install StudySync" at the bottom:
- **Android**: Click "Install" button
- **iOS**: Tap Share → "Add to Home Screen"

## Main Features

### Dashboard
- Overview of all your study data
- Quick access to everything

### Tasks
- Create, edit, complete, delete tasks
- Sort by subject or priority
- Mark as done when completed

### Subjects
- Organize tasks by subject
- Track progress per subject
- Color-coded categories

### Resources
- Store study materials
- Links to websites
- PDFs and notes
- Videos

### Voice Input
- Create tasks by speaking
- Fill in task names with voice
- Hands-free task creation

## Keyboard Navigation

Use Tab to navigate through:
- Dashboard button
- Tasks, Subjects, Resources buttons
- Add Task button
- Language buttons
- All form fields

Press Enter to activate buttons/links.

## Settings

Click your profile (AM - Ana Machava) in the top left to:
- Edit your profile name
- Change your role
- Add email
- Add bio

## Tips

1. **Use Voice for Speed** - Fastest way to add tasks
2. **Color Code Subjects** - Pick different colors for easy sorting
3. **Set Deadlines** - Know when things are due
4. **Use Priorities** - High/Medium/Low helps you focus
5. **Track Progress** - Check subject completion rates

## Mobile Usage

On phones:
- All features work great in single-column layout
- Voice input works perfectly
- Install as app for native experience
- Touch-friendly buttons (big enough to tap)

## Troubleshooting

### Voice Input Not Working
- Check browser (Chrome/Firefox/Safari)
- Allow microphone permission
- Refresh page if stuck

### Tasks Not Saving
- Check internet connection
- Try again in a moment
- Check browser console for errors

### Can't Install App
- Use Chrome for Android
- Use Safari for iOS (iOS 14.5+)
- Check if already installed
